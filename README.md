

# 📢안산 전단지 배포 전문 사이트
안산 지역 기반 전단지 배포 서비스 플랫폼입니다. 지역 소상공인부터 기업까지 효과적인 전단지 마케팅을 위한 최적의 솔루션을 제공합니다.
</br>

## 🚀 주요 기능
- 전화 상담
- 업체 포트폴리오

- 디렉토리 이동

- 의존성 설치
npm install
